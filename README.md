## 🛠️ 技术栈

| 技术 | 说明 |

| Element Plus | UI 组件库 |
| TypeScript | JavaScript 超集 |

## 📌 项目概述

用于餐厅日常营运管理，包括：

- 销售数据统计
- 会员管理、会员权限管理
- 订单管理、优惠管理
- 骑手管理
- 用户管理、餐品管理
- 库存 / 货品管理
- 员工管理、财务管理
- 权限管理、职位管理

---

## 📦 系统模块

### 一、顾客模块

- 用户登录 / 注册
- 在线订餐、订单查询

### 二、数据统计模块

- 营业额统计、订单数统计
- 每日会员售卖、每日值班任务栏

### 三、会员管理

- 会员 CRUD、会员分类

### 四、会员权限管理

- 会员类型 CRUD
- 会员权限配置

### 五、订单管理

- 订单 CRUD
- 订单详情：配送状态、用户信息、日期、状态、实际价格、实付价格、商品价格
- 订单类型：外带 / 堂食 / 外卖
- 骑手信息（外卖订单）
- 优惠券、优惠金额、优惠详情

> **配送状态**：骑手到店时间、店方出餐时间、开始送餐时间（仅外卖订单显示）

### 六、优惠管理

- 优惠 CRUD
- 优惠券码、优惠金额
- 优惠用户类型（会员/普通用户）
- 优惠商品 / 商品类型（特定商品或全品类）
- 优惠门槛（满减条件）、优惠到期时间

### 七、骑手管理

- 骑手 CRUD
- 骑手状态：上线 / 下线（下线后无法接单）
- 骑手账号、密码管理
- 骑手订单改派

### 八、用户管理

- 用户 CRUD
- 用户状态：封禁 / 解禁
- 拥有的优惠券、用户详情信息

### 九、餐品管理

- 餐品 CRUD、餐品分类
- 餐品配方（如：光亮芝麻面包、烤肉酱等半成品组成的配方）
- 餐品状态：锁键 / 解键 / 上架 / 下架

> **餐品状态说明**：锁键 - 用户可见但无法下单；下架 - 完全从菜单中移除

### 十、库存 / 货品管理

- 货品 CRUD、库存管理
- Waste 上报及日志记录

### 十一、员工管理

- 员工 CRUD
- 员工账号、 密码管理
- 员工职位、详细信息

> **职位说明**：员工操作权限（如订单管理、商品锁键等）由职位分配

### 十二、财务管理

- 财务数据统计、财务报表

### 十三、权限管理

- 权限 CRUD
- 职位权限配置

### 十四、职位管理

- 职位 CRUD、职位详情信息

### 十五、餐品批量管理

独立页面批量操作餐品，包含两个 Tab：

- **批量锁键/解键**：通过配方中的半成品或餐品名称搜索批量操作
- **批量上架/下架**：通过配方中的半成品或餐品名称搜索批量操作
